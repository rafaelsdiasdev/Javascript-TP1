## 4. Explique usando 2 exemplos diferentes de pseudocódigo o conceito de repetições e de condicionais. Qual o propósito de cada uma?

### CONDICIONAL:

```javascript
const numero = 10;

if (numero > 5) {
  return `${numero} é maior que 5`;
} else {
  return `${numero} é menor que 5`;
}
```

### REPETIÇÃO:

```javascript
let i = 10;
const b = 20;

while (i < b) {
  i = i + 1;
  console.log(i);
}
```
